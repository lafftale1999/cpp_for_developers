# Guide for using WOKWI
Wokwi is a simulation tool which we can use to run our embedded programs.

## Run your program
1. Open the command palette
This can be achieved through to ways:
* Right click in the code segment - then press command palette
* Left click in the code segment and press F1

2. Upload
Choose the option `Upload Firmware and Start Simulation...`

3. Choose file
Choose your .hex file that you want to run.

You have now uploaded your program and the simulation starts automatically.

## Create a new project
1. Go to: https://wokwi.com/
2. Choose Ecosystem (Arduino)
3. Choose project using Arduino Uno

> You can choose from `Featured Projects` | `Start from scratch` | `Latest Projects`
